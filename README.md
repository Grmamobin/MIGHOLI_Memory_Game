## Installation:

If you want to play it:
you can use `npm` , `yarn` or `bun` to run it.

**NOTE**: Make sure to add `node_modules`

To update and install node_modules run this commands:

```ts
npm install
```
Then :

```ts
npm start
```

or

```ts
yarn start
```
